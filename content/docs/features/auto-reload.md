# Auto-Reload

Galley monitors the open PDF for changes on disk and reloads it automatically
when your TeX engine produces a new version.

The reload preserves:

- The current scroll position
- The current zoom level
- The current page number

This means you can keep Galley open beside your editor and run your build
loop without ever touching the viewer.
